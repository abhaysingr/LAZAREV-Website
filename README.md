# LAZAREV. - Digital Product Design Agency Website

This project is a clone of a "LAZAREV. Digital Product Design Agency" website, focusing on UI/UX and product design for AI & ML businesses. It's a single-page application demonstrating various modern web development techniques and animations.

## Features

*   **Responsive Navigation:** A dynamic navigation bar with hover effects.
*   **Hero Section:** Highlighting "AI & ML Product Design Agency" with animated elements.
*   **Scrolling Animations:** Smooth scrolling and scroll-triggered animations powered by Locomotive Scroll and GSAP.
*   **Interactive Elements:** Hover effects on case study previews and video playback functionality.
*   **Case Studies:** Sections showcasing project case studies like "Accern Rhea" and "AfroTech."
*   **Design Process:** Detailed breakdown of the digital product design process.
*   **Modern Design:** A sleek, dark-themed UI with green accents and modern typography.

## Technologies Used

*   **HTML5:** Structure of the web page.
*   **CSS3:** Styling and layout.
*   **JavaScript (ES6+):** Interactivity and dynamic content.
*   **GSAP (GreenSock Animation Platform):** For powerful JavaScript animations.
*   **Locomotive Scroll:** For smooth scrolling experiences.
*   **Remixicon:** For icons.

## Setup

To run this project locally, simply open the `index.html` file in your web browser.

```bash
# Clone the repository
git clone <repository_url>

# Navigate to the project directory
cd lazarev

# Open index.html in your browser
start index.html # On Windows
open index.html # On macOS
xdg-open index.html # On Linux
```
